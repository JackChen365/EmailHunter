# EmailHunter
### 一个Email 采集的chrome 插件,主要有两个功能
> * 采集当前页面email
> * 采集当前服务器域内email（纯js操作爬采，无后台操作)
<br>

### 使用说明
> 点击弹出Popup会自动采集当前页email信息，点击More,会将当前页作为起始点，遍历所以当前页内的域内连接（不会超出当前服务器域范围，避免无限制的爬取）直到完成为止。<br>
> 中途关闭弹窗，会自动停止后台操作，并清除所有数据。
